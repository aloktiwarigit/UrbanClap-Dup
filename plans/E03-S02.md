# E03-S02 — Customer Catalogue UI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking. WS-A must complete before WS-B; WS-C and WS-D can run in parallel after WS-B.

**Goal:** Replace the `HomeScreen` placeholder with a photo-first catalogue UI — categories grid, service list, and service detail with Trust Dossier preview stub and "Book Now" disabled CTA.

**Architecture:** Clean Architecture. Repository fetches from REST API via Retrofit. Use cases are thin mappers. ViewModels expose `StateFlow<UiState>`. Three Compose screens; navigation wired into existing `MainGraph`.

**Tech Stack:** Kotlin + Compose, Hilt, Retrofit 2 + OkHttp 4 + Moshi, Coil (Compose), Navigation Compose 2.8.9, JUnit 5 + MockK + Robolectric, Paparazzi.

**Sources of truth:** `docs/stories/README.md` §E03-S02, `docs/prd.md` §FR-2.1/FR-2.2

**Known gotchas (read first):**
- `docs/patterns/paparazzi-cross-os-goldens.md` — never run `recordPaparazziDebug` on Windows; delete goldens before push and trigger CI workflow
- `docs/patterns/hilt-module-android-test-scope.md` — JVM unit tests use `mockk()`, NOT `@HiltAndroidTest`
- `docs/patterns/kotlin-explicit-api-public-modifier.md` — every public `class`/`fun`/`val` must carry explicit `public` modifier
- `docs/patterns/firebase-callbackflow-lifecycle.md` — not required here (no async Firebase SDK ops); Coil loads Firebase Storage CDN URLs as plain HTTPS

- [ ] **A-4d: Create `data/catalogue/CatalogueRepositoryImpl.kt`**

```kotlin
internal class CatalogueRepositoryImpl @Inject constructor(
    private val api: CatalogueApiService,
) : CatalogueRepository {

    override fun getCategories(): Flow<Result<List<Category>>> = flow {
        emit(Result.success(api.getCategories().map { it.toDomain() }))
    }.catch { emit(Result.failure(it)) }

    override fun getServicesForCategory(categoryId: String): Flow<Result<List<Service>>> = flow {
        emit(Result.success(api.getServicesForCategory(categoryId).map { it.toDomain() }))
    }.catch { emit(Result.failure(it)) }

    override fun getServiceDetail(serviceId: String): Flow<Result<Service>> = flow {
        emit(Result.success(api.getServiceDetail(serviceId).toDomain()))
    }.catch { emit(Result.failure(it)) }

    private fun CategoryDto.toDomain() = Category(id, name, imageUrl, serviceCount)
    private fun ServiceDto.toDomain() = Service(id, categoryId, name, description, basePrice, durationMinutes, imageUrl, includes, addOns.map { AddOn(it.name, it.price) })
}
```

- [ ] **A-5a: Create `data/catalogue/di/CatalogueModule.kt`**

```kotlin
@Module
@InstallIn(SingletonComponent::class)
public object CatalogueModule {

    @Provides @Singleton
    public fun provideOkHttpClient(): OkHttpClient =
        OkHttpClient.Builder()
            .addInterceptor(HttpLoggingInterceptor().apply {
                level = if (BuildConfig.DEBUG) HttpLoggingInterceptor.Level.BODY
                        else HttpLoggingInterceptor.Level.NONE
            })
            .build()

    @Provides @Singleton
    public fun provideRetrofit(okHttpClient: OkHttpClient): Retrofit =
        Retrofit.Builder()
            .baseUrl(BuildConfig.API_BASE_URL)
            .client(okHttpClient)
            .addConverterFactory(MoshiConverterFactory.create(
                Moshi.Builder().add(KotlinJsonAdapterFactory()).build()
            ))
            .build()

    @Provides @Singleton
    public fun provideCatalogueApiService(retrofit: Retrofit): CatalogueApiService =
        retrofit.create(CatalogueApiService::class.java)

    @Binds @Singleton
    public abstract fun bindCatalogueRepository(impl: CatalogueRepositoryImpl): CatalogueRepository
}
```

Note: Split into `@Module object` (for `@Provides`) and `@Module abstract class` (for `@Binds`) if compiler requires it; merge into companion-object pattern otherwise.

@HiltViewModel
internal class CatalogueHomeViewModel @Inject constructor(
    private val getCategories: GetCategoriesUseCase,
) : ViewModel() {

    private val _uiState = MutableStateFlow<CatalogueHomeUiState>(CatalogueHomeUiState.Loading)
    public val uiState: StateFlow<CatalogueHomeUiState> = _uiState.asStateFlow()

    init {
        viewModelScope.launch {
            getCategories().collect { result ->
                _uiState.value = result.fold(
                    onSuccess = { CatalogueHomeUiState.Success(it) },
                    onFailure = { CatalogueHomeUiState.Error(it.message ?: "Unknown error") },
                )
            }
        }
    }
}
