# E01-S02 Implementation Plan — admin-web skeleton + landing page

> **Retroactive plan.** PR #2 (commit `1236d4c`, merged 2026-04-18) shipped without committing its `plans/E01-S02.md`. Only the brainstorm spec landed at `docs/superpowers/specs/2026-04-18-e01-s02-admin-web-skeleton-design.md`. This plan is reverse-engineered from the merged code + story spec + brainstorm spec to close the BMAD-trail audit gap (issue #123, `docs/audit/story-completeness-2026-04-26.md`). Nothing here drives new work — it documents what was built.

**Goal:** Turn the placeholder agency-baseline scaffold (`client-baseline-nextjs`) into the canonical `homeservices-admin` skeleton: Next.js 15 App Router + TypeScript strict + Tailwind v4 + Storybook + WCAG-AA-clean landing page + green CI, locked to the patterns every subsequent admin-web story (E09-S01..S06, E10-S04, E01-S04, E01-S05) will follow.

**Architecture:** App Router only (no `pages/`). RSC by default with `"use client"` only where needed (theme toggle hook). Tailwind v4 PostCSS-native pipeline (`@import "tailwindcss"` + `@theme` directive in `app/globals.css`). Foundation tokens (color, type, space, radii, elevation, motion) live as CSS custom properties in `globals.css`, mirrored by Tailwind utility classes. Sentry initialised via Next.js 15 `instrumentation.ts` hook with early-return-when-DSN-unset; OpenTelemetry packages **removed** (`@opentelemetry/api`, `@opentelemetry/sdk-node` were not clean no-ops in Next SSR — deferred to a future observability story). Vitest + jsdom + RTL for unit tests; Playwright + axe-core for e2e + a11y. Lighthouse CI for performance budgets. CI workflow at repo-root `.github/workflows/admin-ship.yml` (the placeholder `admin-web/.github/workflows/ship.yml` is dead code — GitHub Actions only discovers root-level workflows). Codex-marker step uses ancestor-check + scope-diff (matches `api-ship.yml`) — never a self-referential SHA-equality check.

**Tech Stack:** Next.js 15 App Router, React 19, TypeScript 5 `strict: true` + `noUncheckedIndexedAccess` + `exactOptionalPropertyTypes`, Tailwind CSS v4 + `@tailwindcss/postcss`, Storybook 8 + `@storybook/nextjs`, `@sentry/nextjs@^8`, Vitest 2 + `@vitest/coverage-v8`, React Testing Library, Playwright + `@axe-core/playwright`, Lighthouse CI 0.14, ESLint 9 flat config, pnpm 9.

**Sources of truth:**
- Story spec: [`docs/stories/E01-S02-admin-web-skeleton-landing-page.md`](../docs/stories/E01-S02-admin-web-skeleton-landing-page.md) — 11 ACs, T1–T11 tasks, 18 anti-patterns
- Brainstorm: [`docs/superpowers/specs/2026-04-18-e01-s02-admin-web-skeleton-design.md`](../docs/superpowers/specs/2026-04-18-e01-s02-admin-web-skeleton-design.md) — 7 questions resolved, 10 disaster fixes slotted
